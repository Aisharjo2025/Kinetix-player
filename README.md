# Kinetix-player
A lightweight android emulator for gaming on linux 

How to install it ?
This emulator works in CachyOS and Linux Mint,I don't know about other distro because I didn't test it on there.
1.Download all files and extract the folder.
2.Open the folder and open terminal there.
3. " chmod +x install.sh " - paste this command and run it
4.then add this " ./install.sh " and again run it 
After this you will see a beautiful installer,go ahead and install it,Enjoy it yourself
